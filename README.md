# LearnWise-AI

An AI-powered learning assistant built using **Python**, **Streamlit**, and the **Groq API**. The application helps students with AI-powered learning through chatbot support, resume analysis, quizzes, learning roadmaps, and personalized course recommendations.

## 🚀 Features

- 🤖 AI Chatbot
- 📊 Student Dashboard
- 📝 Resume Parser
- 🎯 Course Recommendations
- 🧠 Quiz Generator
- 🗺️ Learning Roadmap
- 👤 User Profile
- 💾 Student Database

## 🛠️ Tech Stack

- Python
- Streamlit
- Groq API
- SQLite
- Pandas
- Matplotlib

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/guptanavya0901-ai/LearnWise-AI.git
cd LearnWise-AI
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Create a `.env` file

```env
GROQ_API_KEY=your_api_key
```

### Run the application

```bash
streamlit run app.py
```
